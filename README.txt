Chat Application with AI Assistant
============================

This is a simple chat application that allows you to interact with an AI assistant through a web interface.

Requirements
-----------
- Node.js
- npm (Node Package Manager)
- A modern web browser

Setup
-----
1. Install dependencies:
   ```
   npm install
   ```

2. Start the server:
   ```
   npm start
   ```

3. Open your web browser and go to:
   http://localhost:3000

How to Use
----------
1. Look for the "Chat with Us!" button in the bottom-right corner of your screen
2. Click on it to open the chat window
3. Type your message in the text box
4. Press Enter or click the Send button to send your message
5. The AI assistant will respond in real-time
6. You can minimize the chat by clicking the "Chat with Us!" header again

Features
--------
- Real-time chat with AI
- Clean and simple interface
- Supports Enter key to send messages
- Auto-scrolling chat window
- Minimizable chat window

Note: Make sure both the frontend and backend servers are running for the chat to work properly.
